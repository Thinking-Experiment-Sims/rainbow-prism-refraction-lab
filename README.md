# Rainbow Refraction Lab

Interactive simulation to explain:

- How a primary rainbow forms (refraction -> internal reflection -> refraction)
- Why different colors have different refractive indexes (dispersion)
- Why red is always the outer/top color in the primary rainbow
- How prism and raindrop views compare

## Run locally

From this folder, run any static server, for example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Interaction

- Mouse wheel on canvas: zoom in/out
- Mouse drag on canvas in prism mode: rotate prism
- Teacher Mode: guided 8-cue lecture sequence
